Neocortical circuits underlying visually-guided behaviors in mice

This project was funded by EU Horizon 2020.

Funded by a European Fellowship from the Marie Curie Action of the EU, we sought to

This project came to involve additional approaches for activity across cortical areas, including widefield calcium imaging. As part of the project, we have developed new algorithms for processing electrophysiological and imaging data, and have shared these resources open-source.

We have shared some datasets recorded as part of this project at data.cortexlab.net.

Left, schematic of a Neuropixels probe and image of tip. Right, spiking data recorded from eight probes (over 3000 channels) simultaneously.
